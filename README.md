##### Technology stack:
* HTML, CSS;
* Python 3.8, Django 3.2;
* PostgreSQL.

##### Getting Started
* Create database 'customer_service' in PostgreSQL.
* `$ pip install -r requirements.txt`.
* `$ python manage.py makemigrations`.
* `$ python manage.py migrate`.
* `$ python manage.py runserver`.
